# 📖 Vet Clinic Database <a name="about-project"></a>

**Vet Clinic Database** is a database about animals brought for treatment in the clinic. This project is based on PostgreSQL.

### Key Features <a name="key-features"></a>

This database contains information about
- **Animals History**
- **Animals' Owners**
- **Clinic Employees**
- **Animals' visit records**
